# Quick Start: Your First 3 Agents in 15 Minutes

You don't need to understand the whole system to start using it. Pick three agents. Deploy them. Use them on real work today.

## The Starter Three

These are the agents that change your daily work immediately:

1. **Ink** (Copywriter) — every email, proposal, and client message gets better
2. **Scout** (Design Strategist) — load before any strategy conversation
3. **Donna** (Finance Tracker) — start asking the money questions you've been avoiding

## Step-by-Step: Claude Projects

**Time: 5 minutes per agent**

### Agent 1: Ink

1. Go to [claude.ai](https://claude.ai) → Projects → Create New Project
2. Name it "Ink"
3. Upload to Project Knowledge:
   - `claude/agents/shared-soul.md`
   - `claude/agents/creative/copywriter.md`
4. In Custom Instructions, paste:
   ```
   You are Ink. Read your shared soul and agent files.
   Stay in character. Match every output to the voice
   principles defined in your files.
   ```
5. Start a conversation. Ask Ink to rewrite your last client email.

### Agent 2: Scout

1. Create New Project → Name it "Scout"
2. Upload:
   - `claude/agents/shared-soul.md`
   - `claude/agents/strategy/design-strategist.md`
3. Custom Instructions:
   ```
   You are Scout. Read your shared soul and agent files.
   Stay in character. Think in outcomes, not activities.
   Every recommendation connects to something measurable.
   ```
4. Paste a client brief and ask Scout to identify the top 3 strategic priorities.

### Agent 3: Donna

1. Create New Project → Name it "Donna"
2. Upload:
   - `claude/agents/shared-soul.md`
   - `claude/agents/operations/finance-tracker.md`
3. Custom Instructions:
   ```
   You are Donna. Read your shared soul and agent files.
   Stay in character. Speak in clear numbers, not
   accounting jargon.
   ```
4. Describe your last project and ask Donna to assess profitability.

## What to Do This Week

- **Monday:** Use Ink to draft all external communications
- **Tuesday:** Load Scout before your next strategy call
- **Wednesday:** Ask Donna to review your current project margins
- **Thursday:** Notice what's working and what needs tuning in each agent's personality
- **Friday:** Update the soul files based on what you learned

## Common First-Week Adjustments

**Agent sounds too generic:** Add more to the "What You Are Not" section. Constraints create character.

**Agent doesn't know enough about your work:** Add context to the Project Knowledge. Your past proposals, case studies, writing samples. The more context, the sharper the output.

**Agent gives advice you disagree with:** Good. Update its principles to reflect your actual decision-making. The disagreement is showing you where the soul file needs your real thinking.

**Agent is too verbose:** Add a principle: "Be concise. If you can say it in fewer words, do." Most agents need this.
